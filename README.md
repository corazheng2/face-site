README - FaceTrust 人脸识别系统
系统简介
FaceTrust 是一款基于OpenCV与LBPH算法的人脸识别系统，适用于企业身份核验、婚恋安全匹配、宠物领养回访等多种场景。

功能特点
基于OpenCV计算机视觉库和LBPH(局部二值模式直方图)算法
高精度人脸识别与身份核验
支持多种应用场景
